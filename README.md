# dikueconpub

Public code base for solving economic problems

- `src/logit`: MC code in Python and Futhark for computing some economic measures.

- `src/carmodel`: Matlab and Futhark code for finding the equilibrium of a
  used-car model, using dynamic programming. For a description of the problem
  see [Equilibrium Trade in Automobiles](/src/carmodel/doc/eqb.pdf) and
  [the associated slide set](src/carmodel/doc/eqb_DSE2021.pdf).
